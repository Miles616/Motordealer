# **App Name**: MotorKhan

## Core Features:

- Vehicle Listing: Showcase available vehicles with detailed information such as specifications, photos, and pricing.
- Search and Filtering: Enable users to search and filter vehicles based on various criteria like make, model, year, price range, and body type.
- Vehicle Details Page: A dedicated page for each vehicle providing in-depth information, high-resolution images, and a contact form to inquire about the vehicle.
- Contact Form: Allow users to submit inquiries about specific vehicles or general questions.
- AI-Powered Vehicle Recommendations: Use generative AI tool to suggest vehicles based on user preferences and browsing history, such as budget, preferred body style, or number of seats. Use this information in the query if relevant, ignore it if not relevant

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey trust and sophistication.
- Background color: Light gray (#F5F5F5) for a clean, modern look.
- Accent color: Electric blue (#7DF9FF) for highlighting key elements and calls to action.
- Body font: 'Inter' sans-serif for readability and a modern feel.
- Headline font: 'Space Grotesk' sans-serif for bold headings.
- Use minimalist icons for vehicle features and categories.
- Implement a clean and organized layout with clear visual hierarchy to showcase vehicles effectively. Use grid-based layout for vehicle listings.
